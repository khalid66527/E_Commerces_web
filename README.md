# 🛒 Modern Full-Stack E-Commerce Web Application

একটি আকর্ষণীয়, আধুনিক এবং পরিপুর্ণ ফিচার-সমৃদ্ধ ফুল-স্ট্যাক ই-কমার্স অ্যাপ্লিকেশন। এই প্রজেক্টটিতে ব্যবহারকারীদের উন্নত শপিং অভিজ্ঞতা দেওয়ার পাশাপাশি সিকিউর অথেনটিকেশন, ডায়নামিক ফিল্টারিং, স্ট্রাইপ পেমেন্ট গেটওয়ে ইন্টিগ্রেশন এবং অ্যাডমিন ও ইউজারের জন্য আলাদা শক্তিশালী ড্যাশবোর্ড প্রধান বৈশিষ্ট্য হিসেবে রাখা হয়েছে।

## 🚀 অ্যাপ্লিকেশনের মূল ফিচারসমূহ (Key Features)

### 👤 কাস্টমার/ইউজার ইন্টারফেস ফিচারসমূহ:
1. **নিরাপদ অথেনটিকেশন সিস্টেম:** `Better Auth` ব্যবহার করে সাধারণ ইমেল/পাসওয়ার্ড ও ওয়ান-ক্লিক `Google Social Sign-in` ফিচার।
2. **ডায়নামিক শপ পেজ ও ফিল্টারিং:** পণ্যের নাম সার্চ করা, ক্যাটাগরি, ব্র্যান্ডের নাম কিংবা মূল্যের পরিসীমার (Price Range) ওপর সর্টিং করার আধুনিক সুবিধা।
3. **কার্ট এবং উইশলিস্ট সিস্টেম:** পছন্দসই প্রোডাক্টগুলি সরাসরি কার্টে অথবা উইশলিস্টে যুক্ত করা এবং পরবর্তীতে সেগুলোর সংখ্যা পরিবর্তন বা মুছে ফেলার নিখুঁত ট্র্যাকিং।
4. **সহজ পেমেন্ট ইন্টিগ্রেশন:** নিরাপদ শপিং অভিজ্ঞতা নিশ্চিত করতে `Stripe` এর সাহায্যে রিয়েল-টাইম ব্যাংক ট্রানজেকশন প্রসেসিং।
5. **ক্রয় বিবরণী (Purchase History):** ইউজার প্রোফাইলের আন্ডারে গ্রাহক তার পূর্ববর্তী অর্ডারের সমস্ত পেমেন্ট স্ট্যাটাস ও ট্রানজেকশন রসিদ দেখতে পারবেন।

### 👑 শক্তিশালী অ্যাডমিন ড্যাশবোর্ড ফিচারসমূহ:
1. **অ্যানালিটিক্স ওভারভিউ:** টোটাল সেলস রেভিনিউ, সক্রিয় ইউজারের সংখ্যা এবং মোট পণ্যের স্ট্যাটাস গ্রাফ ও ভিজ্যুয়াল কার্ডে প্রদর্শন।
2. **প্রোডাক্ট ইনভেন্টরি কন্ট্রোল:** নতুন প্রোডাক্ট আপলোড করা (`Post a Product`), পণ্যের ডিটেইলস এডিট করা কিংবা ডাটাবেজ থেকে মুছে ফেলার প্যানেল।
3. **ইউজার কন্ট্রোল প্যানেল:** রেজিস্টার্ড সকল ব্যবহারকারীদের তালিকা দেখা, তাদের ইমেইল ও রোল আপডেট (যেমন: `User` থেকে `Admin` এ রূপান্তর) করার চমৎকার অ্যাডমিন প্রিভিলেজ।
4. **লেনদেন বিবরণী (View Transactions):** স্ট্রাইপ পেমেন্টের মাধ্যমে হওয়া সকল সফল পেমেন্ট বা অসম্পূর্ণ ট্রানজেকশনের রেকর্ড ভেরিফাই করার সুবিধা।

### ধাপ ২: এনভায়রনমেন্ট ভেরিয়েবল সেটআপ (Environment Setup)

উভয় প্রজেক্টের রুট ডিরেক্টরিতে যথাক্রমে `.env` ফাইল তৈরি করুন এবং নিচের কী-গুলি ইনপুট দিন:

#### **১. ফ্রন্টএন্ড `.env` কনফিগারেশন:**
`E_Commerces_web` ফোল্ডারে একটি `.env` ফাইল বানিয়ে নিচের কোডটি লিখুন:
```env

MONGO_DB_URI=আপনার_মঙ্গোডিবি_কানেকশন_ইউআরআই
AUTH_DB_NAME=ডাটাবেজের_নাম

GOOGLE_CLIENT_ID=গুগল_ক্লায়েন্ট_আইডি
GOOGLE_CLIENT_SECRET=গুগল_ক্লায়েন্ট_সিক্রেট

NEXT_PUBLIC_API_URL=http://localhost:5000
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=স্ট্রাইপ_পাবলিশেবল_কী
STRIPE_SECRET_KEY=স্ট্রাইপ_সিক্রেট_কী
JWT_SECRET=আপনার_জেডব্লিউটি_সিক্রেট_কী
```

## 🧱 কিভাবে প্রজেক্টে নতুন পেজ এবং কম্পোনেন্ট তৈরি করবেন? (File Creation Guideline)

Next.js-এর ফাইল-ভিত্তিক রাউটিং মেকানিজম বা অ্যাপ রাউটার (App Router) মেনে সহজেই যেকোনো নতুন পেজ বা UI কার্ড যুক্ত করতে পারবেন। নিচে তার কিছু নিয়ম দেওয়া হলো:

### ১. নতুন কোনো পেজ তৈরি করতে চাইলে (Creating a New Page/Route)
Next.js অ্যাপ রাউটারে একটি ফোল্ডার মানেই একটি নির্দিষ্ট রাউট।
* মনে করুন আপনি নতুন একটি **FAQ** পেজ তৈরি করবেন (`/faq` রাউট)।
* প্রথমে `src/app/` ডিরেক্টরিতে যান এবং `faq` নামে একটি নতুন ফোল্ডার তৈরি করুন।
* সেই `faq` ফোল্ডারের ভেতরে `page.tsx` নামে একটি ফাইল তৈরি করুন এবং নিচের কোডটি লিখুন:
  ```tsx
  // src/app/faq/page.tsx
  import React from 'react';

  export default function FAQPage() {
    return (
      <main className="max-w-4xl mx-auto p-8 my-10">
        <h1 className="text-3xl font-bold text-gray-800 mb-6">Frequently Asked Questions (FAQ)</h1>
        <div className="space-y-4">
          <div className="border p-4 rounded-lg">
            <h3 className="font-semibold text-lg">১. পণ্য ডেলিভারি পেতে কতদিন সময় লাগে?</h3>
            <p className="text-gray-600 mt-2">সাধারণত অর্ডার নিশ্চিত করার ২-৩ কার্যদিবসের মধ্যে ডেলিভারি সম্পন্ন হয়।</p>
          </div>
        </div>
      </main>
    );
  }
  ```
* ফাইলটি সেভ করার সাথে সাথে ব্রাউজারে `http://localhost:3000/faq` ইউআরএল এ আপনার নতুন পেজটি দেখতে পাবেন।

### ২. নতুন রিইউজেবল ইউআই কম্পোনেন্ট তৈরি করতে চাইলে (Creating Reusable Components)
যদি আপনার এমন কোনো কার্ড বা বাটন তৈরি করতে হয় যা একাধিক পেজে দরকার:
* প্রথমে `src/components/` ডিরেক্টরিতে যান।
* আপনার কম্পোনেন্টের নাম অনুসারে ফাইল তৈরি করুন, যেমন: `DiscountBadge.tsx`।
* ফাইলের ভেতর কাস্টম কোড ডিফাইন করে এক্সপোর্ট করুন:
  ```tsx
  // src/components/DiscountBadge.tsx
  import React from 'react';

  interface DiscountBadgeProps {
    percentage: number;
  }

  export default function DiscountBadge({ percentage }: DiscountBadgeProps) {
    return (
      <span className="bg-red-500 text-white text-xs font-bold px-2 py-1 rounded-full animate-pulse">
        {percentage}% OFF
      </span>
    );
  }
  ```
* এবার যেকোনো পেজে এই কম্পোনেন্টটি ইম্পোর্ট করে সহজেই ব্যবহার করুন:
  ```tsx
  import DiscountBadge from '@/components/DiscountBadge';
  ```
